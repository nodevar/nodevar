## About Me

I work at **Netcloud AG** in Switzerland, building public cloud services on Azure and AWS. I spent a decade in network engineering before moving to cloud — that background shapes how I think about infrastructure.

- Building **Azure landing zones and platforms** in Bicep and Terraform — production workloads, not lab demos
- Automating workflows with **Claude Code, MCP, and AI agents** — from Notion integrations to full content pipelines
- Writing **infrastructure as code** that actually ships to production
